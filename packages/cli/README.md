# @agent-seo/cli

CLI audit tool for AI visibility scoring. Check how well your website serves AI crawlers.

## Usage

```bash
npx @agent-seo/cli audit https://example.com
```

Get an AI Visibility Score (0-100) with actionable recommendations:

- Bot detection and response analysis
- Markdown quality assessment
- `/llms.txt` presence and validity
- Response headers check
- Structured data extraction

## Install (optional)

```bash
npm install -g @agent-seo/cli
agent-seo audit https://example.com
```
