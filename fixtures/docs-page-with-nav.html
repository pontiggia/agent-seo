Getting Started

MyLib is a lightweight utility library for building web applications. It provides a set of composable functions that simplify common tasks.

Installation

Install MyLib using your preferred package manager:

npm install mylib
# or
yarn add mylib
# or
pnpm add mylib

Quick Start

Import the functions you need and start using them:

import { createApp, defineConfig } from 'mylib';

const config = defineConfig({
  debug: true,
  plugins: ['auth', 'logger'],
});

const app = createApp(config);
app.start();

Configuration Options

MyLib accepts the following configuration options:

OptionTypeDefaultDescription
debugbooleanfalseEnable debug logging
pluginsstring[][]Plugins to load
portnumber3000Server port
hoststringlocalhostServer host
